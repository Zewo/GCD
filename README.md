**GCD** is a Grand Central Dispatch wrapper.

## Features

- [x] Queues
- [x] Semaphore
- [x] IO Channel

## Installation

```swift
import PackageDescription

let package = Package(
    dependencies: [
        .Package(url: "https://github.com/Zewo/GCD.git", majorVersion: 0, minor: 4)
    ]
)
```
